#  Currency Converter



A live currency converter built with Streamlit that converts currencies in real time with current and inverse rates.

---

 Features

- Convert any amount from one currency to another  
- Dropdowns with all supported currencies  
- Shows current exchange rate and inverse rate  
- Clean, easy-to-use interface

# Installation (Local Setup)
1.Clone the repository
  ```git clone https://github.com/tadiwarodney/currency-converter.git```
2.Install dependencies
  ```pip install -r requirements.txt```
3.Create a .env file in the root directory and add
  ```EXCHANGE_API_KEY= your_api_key```
4.Run the application
  ```streamlit run app.py```
